* 조건문

   * ```if-else``` 구조를 사용하여 점수 범위에 따라 등급 판별

### 2. 구매자가 장바구니에 담은 상품 목록을 볼 수 있는 기능
1) 사용 기능
* 컬렉션
  *  ```List```, ```Map```
* 반복문
  *  ```for-in```
* 조건문
  *  ```if```, ```else```

2) 주요 특징
   * ```List```(장바구니)와 ```Map```(가격)을 정의
   * ```.isEmpty```를 사용하여 장바구니가 비었을 경우 예외 처리
   * 반복문 처리: for-in
   * Map 탐색: ```price.containsKey(item)```을 사용하여 가격 정보 유무 확인
   * ```if``` 조건문을 사용하여 2만원 이상 10% 할인율 적용
   * ```double```을 사용하여 소수점 계산 처리
   * ```.round()```로 정수 형태로 출력 처리
  
[출력 값]
  
```
장바구니에 40000원 어치를 담으셨네요!
할인 금액: 4000원
최종적으로 결제하실 금액은 36000원 입니다! 
```

### 2. 구매자가 장바구니에 담은 상품 목록을 볼 수 있는 기능
1) 사용 기능
  * 컬렉션
    *  ```Set```
  * 클래스
    *  ```Random```
  * 반복문
    *  ```While```
* 조건문
  *  ```if```, ```else```
* 초기화
  *  ```.clear()```

2) 주요 특징
   * ```Set```을 통해 로또 번호 중복 방지
   * ```Random```: import 'dart:math';를 통해 가져와 랜덤 숫자 생성
   * ```while```문을 사용하여 로또 번호 갯수가 6이 될 때까지 번호를 생성
   * 당첨번호 Set을 지정해주고, 랜덤 번호와 당첨 번호를 ```.intersection()``` 를 통해 교집합을 찾아냄
   * if 조건문을 통해 교집합 ```set```의 갯수를 ```count```로 세고, 갯수에 따라 당첨 범위를 지정 후 ```print``` 로 등수 출력
   * ```.clear```를 통해 랜덤 번호 초기화
  
 [출력 값]
  
```
발급한 로또 번호: {26, 30, 5, 7, 3, 16}
당첨 여부: 당첨 실패!
현재 발급한 로또 번호: {}
```
